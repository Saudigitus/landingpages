SNIS

Sistema Nacional de Informação Sanitária

Desenvolvido com base na plataforma DHIS2, de código aberto flexível baseado na Web que coleta, valida, analisa e apresenta os dados estatísticos agregados, adaptado às actividades de gestão integrada de informação em saúde.

Acesso às plataformas

O SNIS é um sistema de gestão de informação sanitária baseado no DHIS2, de código aberto flexível baseado na Web que coleta, valida, analisa e apresenta os dados estatísticos agregados, adaptado às actividades de gestão integrada de informação em saúde. Funciona como um sistema online e também intermitente com as funcionalidades apresentadas a seguir.

Dados Agregados

Subsistema para recolha e análise de dados, através de formulários de dados agregados, que resulta da combinação de varios itens como sexo, idade,etc.

Dados Individuais

Subsistema de gestão e captura de eventos que ocorrem na comunidade e o registo feito durante o processo de atendimento ao paciente, com base nos processos individuais dos mesmos.

Plataforma de Serviços de Interoperabilidade

Plataforma de serviços web que permite a ligação directa e segura de comunicação de dados e informações, entre sistemas informáticos de gestão de dados do Ministério da Saúde, de forma transparente e simplificada.

Plataformas de Treinamento

Dados agregados

Este módulo permite a recolha de dados através de formulários de dados agregados. A desagregação pode ser uma combinação de vários itens ( idade, sexo, etc) dependendo das necessidades de cada formulário.

Treinamento
Dados Individuais

Neste módulo são capturados os vários eventos que ocorrem durante o processo de atendimento ao paciente. A captura é efectuada com base nos processos individuais, com a possibilidade de rastrear o tratamento do paciente.
